10 Useful Tags

  1. <h1>, <h2> ... <h6> : Header tags can be used to create text hierarchy in a document.
  2. <!DOCTYPE html> : Used to declare your document's intention to be rendered as HTML5.
  3. <head> : Contains meta information about your document.
  4. <body> : Contains content to be rendered to the page by the browser.
  5. <div> : Used to create a semantic division between elements.
  6. <a> : Create a hyperlink to another resource.
  7. <img /> : Display an image given a src attribute.
  8. <nav> : Semantic alternative to an unordered list for navigation. (New in HTML5.)
  9. <em> : Emphasis - Semantic alternative to <b> or <i>, the characteristics of which should be relegated to your stylesheets.
  10. <script> : Allows for linked JS to introduce interaction and behavior to your document.

Not To Be Used:

  • <b>
  • <big>
  • <br />
  • <i>
  • <small>
